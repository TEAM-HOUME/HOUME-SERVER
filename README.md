나만의 개성과 취향이 담긴 공간을 만들어가는 특별한 경험을 제공하는 서비스입니다.

---

## 📌 서비스 소개

HOUME는 사용자가 자신만의 개성 있는 공간을 디자인하고 꾸밀 수 있도록 도와주는 AI 기반 인테리어 플랫폼입니다.
